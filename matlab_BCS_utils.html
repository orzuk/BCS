  • Overview:
    This page accompanies the paper:
    Bacterial Community Reconstruction Using Compressed Sensing, A. Amir and O. Zuk, Journal of Computational Biology, 18(11):1-20 (2011)
    The paper presents the Bacterial Compressed-Sensing (BCS) algorithm. The source code is available as a Matlab package below.
    The package contains functions for mixture chromatogram simulations, pre-processing and reconstruction.
    Additional functions are used for handling the database of s16-RNA gene, and for displaying results.
    Please see the file "readme.txt" for a coprehensive description of the package.
    In addition, a few data files related to the project are available for download here .
  • About:
    BCS is a collaboration between the Broad Institute and the Weizmann Institute of Science.
  • Installation:
    To install the package simply download the file BCS.tgz to a directory of your choice, extract it, and add to your Matlab path.
    The package uses the GPSR algorithnm, which is available through here .
    For questions, suggestions, bug-reports etc. please contact Amnon Amir (amnon.amir@weizmann.ac.il) or Or Zuk (orzuk@broadinstitute.org).
    Below we list all functions available in the package with their description:
BinChromatogram.m  -  Bins the chromatogram (from scfread) into constant sized bins following
CompareSets5.m  -  Compare reconstructed mixture to original mixture and calculate statistics
GPSRChromatogram.m  -  Run the GPSR algorithm with chromatogram binning predicted for each sequence
GPSRSimulation.m  -  Perform reconstruction of a simualted mixture using GPSR
GetRandomMixtureVec2.m  -  Get the number of creatures in the mixture and the total number creatures
LoadFasta.m  -  Read a fasta file
MultisequenceDemo.m  -  Load the reduced (distance<2 joined together), primer aligned, data
PredictFullSeq.m  -  Predict full sequence chromatogram, and divide it into bins
PredictFullSeqSqr.m  -  Predict full sequence chromatogram, and divide to bins using quadratic peak position correction
PredictFullSeqSqrVar.m  -  Predict full sequence chromatogram, and divide to bins using quadratic peak position correction
SensSel2.m  -  Compute RMSE/sensitivity/selectivity statistics for various thresholds
SeqDist.m  -  Compute Hamming distance between two sequences.
TestBinning.m  -  Compare two binned chromatograms
load20srna.m  -  One time script: convert database to matlab format
plotcartogram.m  -  Plots a 4*L matrix as a 'cartogram'
reconstructmixture.m  -  Main reconstrubtion funcion for mixture chromatogram
reduceset.m  -  Find homologus sequences and cluster them.
simulatemetagenomemixture.m  -  Generate a random species mixture and calculate the PSSM
testmetagenomefunctions.m  -  A script for testing various functions of the project.
ReduceSetFast.c  -  Cluster database sequences - unite sequences with few mismatches
PeakHeightStat.pl  -  Collect Statistics for Peak Heights
PeakPosStat.pl  -  Collect Statistics for Peak Positions
